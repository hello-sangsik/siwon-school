# be able to, be going to

*조동사가 연달아 나올수 없을 때 쓴다*
*be able to 와 be going to 뒤에는 동사원형이 와야한다*

**will can go to => will be able to**

can read a book.
be able to read a book.
He can come.
He is able to come.

I will be able to study today.
If my mother comes, I will be able to have a free time.
If I drive every day in seoul, I will be able to drive well.
He will be able to go alone.

*사실적인 예정과 계획을 말하는 경우 will 대신 be going to 를 쓰는 경우가 많음*
*말하기에선 be going to 를 줄여서 be gonna로 말할 수 있다*

I will drive my car.
I am going to drive my car.
I'm going to leave my bag here.
I will leave my bag here.

**take this chance** 이 기회를 잡다

*be going to 의 부정은 be 뒤에 not 을 붙여 만든다*
I'm not gonna make.
I'm not gonna do.

*절대 ~안 할 것이다는 be 뒤에 never를 붙인다*
I'm never gonna do this.

You might be able to~ 
너 할 수 있을지도 몰라 (100% 확신이 아닌 추측)

should be able to 
 ~할 수 있게 되어야만 한다 (권유의 느낌)

have to be able to 
 ~할 수 있게 되어야만 한다 (강도 높음)

must be able to 
 ~할 수 있게 되어야만 한다 (강도 높음)

**speak up** 목소리를 내다/의견을 내다

You have to be able to confront.
